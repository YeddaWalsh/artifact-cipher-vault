# Changelog

All notable changes to the Artifact Cipher Vault project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with Hardhat and FHEVM integration
- Core smart contract for encrypted counter operations
- Next.js frontend with wallet connectivity
- Comprehensive test suite
- CI/CD pipeline with GitHub Actions
- Development setup scripts for multiple platforms
- Documentation and troubleshooting guides

### Changed
- Improved FHE operations with enhanced encryption
- Enhanced user interface with better accessibility

### Fixed
- Contract deployment issues
- Frontend connection problems
- Build and testing pipeline errors

## [1.0.0] - 2025-11-15

### Added
- Basic project structure and configuration
- Smart contract development environment
- Frontend application framework
- Initial documentation

### Security
- Implemented FHE-based data encryption
- Added access control mechanisms
- Secure key management for decryption
